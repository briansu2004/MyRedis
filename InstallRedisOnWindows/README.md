# Install Redis on Windows 10

## Set up

```
https://github.com/microsoftarchive/redis/releases
3.2.100
Redis-x64-3.2.100.zip
cd C:\Apps\Redis-3.2.100
redis-server.exe
redis-cli.exe
Port: 6379
```

```
https://github.com/tporadowski/redis/releases
5.0.10
Redis-x64-5.0.10
cd C:\Apps\Redis-5.0.10
redis-server.exe
redis-cli.exe
Port: 6379
```

```
docker pull redis
```
